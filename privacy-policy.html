Privacy Policy for Meeting Scheduler

Last updated: January 22, 2026

Overview

Meeting Scheduler is a Pumble add-on that helps teams schedule meetings with calendar integration and email invitations.

Data Collection

We collect and process only the data necessary to provide meeting scheduling functionality:

  • Meeting details (title, description, date, time, location)
  • Attendee email addresses
  • User information from Pumble (name, email, user ID)

Data Usage

Your data is used exclusively to:

  • Create and send calendar invitations (ICS files) via email
  • Sync meetings with CalDAV-compatible calendars
  • Post meeting summaries in Pumble channels
  • Send meeting updates and cancellations

Data Storage

  • Meeting data is stored temporarily in memory during operation
  • No long-term database storage of meeting content
  • Calendar events are stored on your configured CalDAV server (if enabled)
  • Email invitations are sent via your configured SMTP server

Data Sharing

We do NOT:

  • Share your data with third parties
  • Sell or rent your information
  • Store data on external servers beyond your configuration

Data is only transmitted to:

  • Your configured email server (SMTP) for sending invitations
  • Your configured CalDAV server for calendar synchronization
  • Pumble API for posting meeting summaries

User Rights

You can:

  • Cancel meetings at any time
  • Request deletion of meeting data
  • Disconnect the add-on from your workspace
  • Access and export your meeting data

Security

  • All connections use encrypted protocols (HTTPS, TLS/SSL)
  • Credentials are stored securely in environment variables
  • No passwords or sensitive data stored in logs
  • Meeting data is not retained after processing

Third-Party Services

This add-on integrates with:

  • Your email server (via SMTP) - for sending calendar invitations
  • Your CalDAV server (optional) - for calendar synchronization
  • Pumble API - for messaging functionality

Each service has its own privacy policy that applies to data processed by them.

Data Retention

Meeting data is retained only as long as necessary to fulfill the scheduling request. Once a meeting is created and invitations are sent, no personal data is permanently stored by the add-on.

Compliance

This add-on complies with:

  • General Data Protection Regulation (GDPR)
  • Standard data protection practices
  • RFC 5545 (iCalendar) standards for calendar data

Changes to This Policy

We may update this privacy policy from time to time. Changes will be posted on this page with an updated "Last updated" date.
